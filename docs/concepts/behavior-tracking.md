# Behavior Tracking

> Observation and analysis of user interaction patterns within the platform to derive engagement insights, learning preferences, and capability signals.

## Overview

Behavior Tracking captures meaningful interaction events — how users approach assessments, their problem-solving strategies, time allocation patterns, and help-seeking behavior. These behavioral signals complement direct capability measurements to build a richer user model.

## Tracked Dimensions

| Dimension | Signals | Insight |
|---|---|---|
| **Pacing** | Time per question, session duration, break frequency | Processing style, fatigue detection, optimal session length |
| **Strategy** | Answer revision rate, skip patterns, option elimination | Confidence calibration, decision-making style |
| **Help Seeking** | Hint usage, resource access, mentor queries | Learning preference, autonomy level, knowledge gaps |
| **Exploration** | Navigation depth, feature discovery, content browsing | Curiosity score, engagement depth |
| **Consistency** | Performance variance across sessions, days, topics | Reliability, external factor correlation |

## Privacy & Ethics

Behavioral data is processed with strict privacy boundaries:
- Raw event data is not stored with personal identifiers beyond 30 days
- Aggregated behavioral patterns are anonymized for analysis
- Users can view and delete their behavioral profile
- No behavioral data is shared externally

## Integration

Behavioral signals feed into the [AI Mentor](ai-mentor.md) for adaptive coaching, [Analytics](analytics.md) for engagement dashboards, and [Confidence Tracking](confidence-tracking.md) for measurement reliability.
